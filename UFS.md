UFS is a generic term used by [TSK](/The-Sleuth-Kit/) to refer to the UFS1 and UFS2 file systems.  They are commonly found on BSD / Solaris systems and are supported by TSK. 

For more information, refer to:
* [File System Forensic Analysis Book](http://www.digital-evidence.org/fsfa/)
* [Forensics Wiki  Entry](http://www.forensicswiki.org/wiki/Ffs)
* [Wikipedia Entry](http://en.wikipedia.org/wiki/Unix_File_System)
