blkls lists details about file system [data unit](/Data-unit/)s.  In its default mode, it outputs the unallocated data unit contents to STDOUT.  It can also list the details about which are allocated and which are not. It used to be called dls. 

* [Automatically Updated man Page](http://www.sleuthkit.org/sleuthkit/man/blkls.html)
