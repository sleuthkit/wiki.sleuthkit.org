tsk_gettimes will extract the metadata from the file systems in an image.  The results will be a body file that can be used by [mactime](/Mactime/) to make a [timeline](/Timeline/).  This tool is equivalent to running [fls](/Fls/) '-m' on each file system in an image. 

* [Automatically Updated man Page](http://www.sleuthkit.org/sleuthkit/man/tsk_gettimes.html)
