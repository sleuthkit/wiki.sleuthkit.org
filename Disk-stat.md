## disk_stat
Back to [Help Documents](/Help-Documents/)

disk_stat was a Linux-only tool that uses ATA commands to query an ATA disk to determine if it has a Host Protected Area (HPA). 
This tool was removed from TSK.  Use hdparam instead. 

* [Sleuth Kit Informer #17](http://www.sleuthkit.org/informer/sleuthkit-informer-17.html#hpa)
* [Automatically Updated man Page](http://www.sleuthkit.org/sleuthkit/man/disk_stat.html)
