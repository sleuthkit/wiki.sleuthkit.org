ExtX is a generic term used by [TSK](/The-Sleuth-Kit/) to refer to the Ext2 and Ext3 file systems.  They are commonly found on Linux systems and are supported by TSK. 

For more information, refer to:
* [File System Forensic Analysis Book](http://www.digital-evidence.org/fsfa/)
* [Forensics Wiki Ext2 Entry](http://www.forensicswiki.org/wiki/Ext2)
* [Forensics Wiki Ext3 Entry](http://www.forensicswiki.org/wiki/Ext3)
* [Wikipedia Ext2 Entry](http://en.wikipedia.org/wiki/Ext2)
* [Wikipedia Ext3 Entry](http://en.wikipedia.org/wiki/Ext3)

Documents used during development of ExtX support:
* [Design and Implementation of the Second Extended File System](http://web.mit.edu/tytso/www/linux/ext2intro.html) (Card, Ts'o, and Tweedie)
* [Linux EXT2FS Undeletion mini-HOWTO](http://en.tldp.org/HOWTO/mini/Ext2fs-Undeletion.html) (Aaron Crane)
* [EXT3](http://olstrans.sourceforge.net/release/OLS2000-ext3/), Journaling Filesystem (Tweedie)
