The Sleuth Kit (TSK) is a digital forensics library and collection of command line tools that enable you to analyze disk images.  The [TSK Framework](/TSK-Framework/) makes it easier to build end-to-end digital forensics solutions.  TSK can be used in isolation, with the [Autopsy](/Autopsy/) user interface, or with one of the many [Tools Using TSK or Autopsy](/Tools-Using-TSK-or-Autopsy/).

You can get the official list of features at the [sleuthkit.org](http://www.sleuthkit.org/sleuthkit/desc.php) site.

## Capabilities
A summary of the tools contained in TSK can be found on the [TSK Tool Overview](/TSK-Tool-Overview/) page. Currently, TSK supports the following file systems:
* [EXT2, EXT3, EXT4](/ExtX/)
* [FAT](/FAT/), [exFAT](/ExFAT/)
* [HFS](/HFS/)
* [ISO 9660](/ISO9660/)
* [NTFS](/NTFS/)
* [UFS 1, UFS 2](/UFS/)
* [YAFFS2](/YAFFS2/)
