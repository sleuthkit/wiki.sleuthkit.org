# Tools and Libraries that are used by [The Sleuth Kit](/The-Sleuth-Kit/)
(in alphabetical order)
* [AFFLib](http://www.afflib.org/) (AFF image format support)
* [ftp://ftp.astron.com/pub/file/ file] (detects file type)
* [libewf](http://www.uitwisselplatform.nl/projects/libewf/) (EnCase / Expert Witness image format support)

# File Hash Databases
(in alphabetical order)
* [CyberAbuse Rootk(it)ID project](http://rk.cyberabuse.org/?page=credits)
* [Hash Keeper](http://www.hashkeeper.org/)
* [KnownGoods](http://www.knowngoods.org/)
* [NIST NSRL SW Fingerprint Database](http://www.nsrl.nist.gov/)
* [RPM](http://www.rpm.org/) Use on Linux systems with '-V -a' to identify binaries that are different than the local database says
* [Solaris Fingerprint Database](http://sunsolve.Sun.COM/pub-cgi/fileFingerprints.pl)
