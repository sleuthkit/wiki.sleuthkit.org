img_cat outputs  the  contents of an image file.  Image files that are not raw will have embedded data and metadata.  img_cat will output only the  metadata.  This allows you to convert an embedded format to raw or to calculate the MD5 hash of the data  by  piping  the  output  to  the appropriate tool.
