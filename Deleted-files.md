Deleted files are those that have been deleted by a user, but whose content can still be recovered.  There are many different levels of deleted files.  Some still have names that can point to file content in [data units](/Data-units/).  Others have no name, but still have the file content layout.  Others have no file system-level information, but can be recovered using [carving](/Carving/) techniques.
