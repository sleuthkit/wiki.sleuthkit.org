There are a few ways in which you can communicate about [TSK](/The-Sleuth-Kit/) and [Autopsy](/Autopsy/).  The [sleuthkit-users](/Sleuthkit-users/) and [sleuthkit-developers](/sleuthkit-developers/) e-mail lists exist to allow communication about using and developing the software. There are [trackers](/Trackers/) to keep track of bugs and feature requests. This wiki exists to allow people to contribute documentation that they have created or edited.
