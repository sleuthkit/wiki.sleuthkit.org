Block Address is a term that is used in [TSK](/The-Sleuth-Kit/) as generic word for file system-specific terms used to describe groups of consecutive [sectors](/Sectors/).  Each file system has its own term for his concept:
* [FAT](/FAT/): Clusters (NOTE: See [FAT Implementation Notes](/FAT-Implementation-Notes/))
* [NTFS](/NTFS/): Clusters
* [UFS](/UFS/): Blocks and fragments
* [ExtX](/ExtX/): Blocks and fragments
* [HFS](/HFS/): Blocks
* [YAFFS2](/YAFFS2/): Blocks
